ArchaeoCode

Paste a public GitHub repository URL. We clone it, mine the git history, and generate a full archaeology report — dead files, timelines, intent breakdown, and more.

Public repos only. Analysis may take 30–90 seconds.

What you get

  • Dead-file verdicts with confidence scores and orphan detection
  • Architecture timeline — month-by-month file growth and shrink
  • Deleted-feature history for files that lived then disappeared
  • Commit intent breakdown (feature, bugfix, refactor, and more)
  • Most bug-prone files ranked by fix-commit frequency
